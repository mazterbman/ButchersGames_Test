# 🏃ButchersGames_Test (Unity)

Клон кор-механики гиперказуального раннера [Rich to Poor](https://play.google.com/store/apps/details?id=com.ohmgames.richtopoor): персонаж бежит по уровню, собирает пикапы, которые двигают шкалу «богатство ↔ бедность», а внешний вид персонажа меняется в зависимости от её значения. Цель — добежать до финиша.

Задача тестового: максимально точно воспроизвести управление, UI, анимации и эффекты референса и собрать первый уровень.

## 🎬 Демонстрация

▶️ **Геймплей:** [https://youtu.be/VIDEO_ID](https://github.com/user-attachments/assets/9659acb8-0846-4e8f-8a30-2a6787632689)

Показан первый уровень и переключение на следующий.

## 🎮 Управление

Управление реализовано **1:1 с референсом**: удержание и движение пальца/курсора по горизонтали смещает персонажа по дорожке, бег вперёд — автоматический.

## ✨ Что реализовано

**Сцена геймплея**
- Камера, освещение
- Игровой UI (без магазина и меты)
- Туториал в начале уровня

**Game Loop**
- Стартовый UI → старт игры → прохождение уровня → Победа / Поражение
- Менеджер уровней (переключение уровней)

**Элементы уровня**
- Препятствия
- Пикапы: монетки и алкоголь
- Области с флагами
- Ворота
- Область финиша

**Игровые особенности**
- Шкала, изменяемая пикапами
- Смена модели персонажа в зависимости от значения шкалы

**Полишинг**
- Визуальные эффекты
